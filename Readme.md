# LearnFlow Dashboard

A modern student learning dashboard built with Next.js, TypeScript, Tailwind CSS, Framer Motion, and Supabase.

## Live Demo

🔗 https://student-learning-dashboard-h1ubvd7il-riverfy.vercel.app

## Features

* Responsive dashboard UI
* Bento Grid layout
* Course progress tracking
* Learning activity visualization
* Framer Motion animations
* Supabase integration
* Loading and error states
* Dark mode design

## Tech Stack

* Next.js 15
* TypeScript
* Tailwind CSS
* Framer Motion
* Supabase
* Lucide React
